# Hotstar-pipeline-automation
This project demonstrates a complete end-to-end CI/CD pipeline for a Java Maven web application — a Hotstar clone — using Jenkins Declarative Pipeline. The pipeline automates every phase of the software delivery lifecycle: from source code checkout to build, test, quality analysis, artifact storage, and deployment.
The goal is to showcase industry-standard DevOps practices including continuous integration, static code analysis, artifact management, and automated deployment.
